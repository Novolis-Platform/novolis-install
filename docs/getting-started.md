# Getting started

## Install the tool

```bash
dotnet tool install --global Novolis.Install
novolis --help
```

## Commands

| Command | Status |
|---------|--------|
| `novolis search [term]` | Lists registry entries |
| `novolis info <id>` | Package summary (manifest TBD) |
| `novolis install <id>` | Stub |
| `novolis update [id]` | Stub |
| `novolis list` | Shows local state path |
| `novolis remove <id>` | Stub |
| `novolis doctor` | Environment and path check |
